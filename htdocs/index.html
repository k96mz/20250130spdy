Delivery of pbf tiles as static hosting

You have "htdocs" folder for static hosting. If your vector tiles(pbf) are stored in "htdocs" folder, the VT can be accessible at:
http://localhost:3000/VTpractice/1/1/1.pbf

For example, you can get pbf tile at:
./VTpractice/1/1/1.pbf

Hosting Mbtiles

If you have mbtiles/VTpracticegzip.mbtiles for hosting mbtiles, the VT can be accessible at:
http://localhost:3000/VT/zxy/VTpracticegzip/1/1/1.pbf

For example, you can get pbf tile at:
./VT/zxy/VTpracticegzip/1/1/1.pbf


Check the Map with the VT

Regarding static folder, you can check the map with this HTML file:
./map/mapO-pbf.html

for HTTPS access(This doesn't work maybe because we use spdy)
./map/mapOs-pbf.html

Regarding mbtiles, you can check the map with this HTML file:
./map/mapO-mbtiles.html

for HTTPS access(You may reload sometimes to show the map)
./map/mapOs-mbtiles.html
